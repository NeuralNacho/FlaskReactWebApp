Statistics Project Information

Here I explain what this project is about and break down some of the statistics more simply than the report if you are unfamiliar with linear models. This is a computing project I completed at university in my third year and is one of four such projects I did that year. I used R to run the calculations and generate the graphs. Project link here.

The project has a list of questions which challenge you to write programs which give new insights about variable selection. Variable selection can be best explained with an example. For instance, at the end of the project, we are given a dataset on Prostate cancer as well as 8 covariates including age, weight etc. The aim is to choose choose the covariates which actually impact prostate cancer and make a model to predict its likelihood based on the data provided.

The 'bias-variance tradeoff' describes how developing a model based on limited data can lead to the model being overfitted to that data, potentially reducing its predictive accuracy on new data. I investigate ways to mitigate this overfitting by using different methods of variable selection. I also look at the computational complexity of these methods which is affected by matrix sparsity and the number of subsets of different variables.
